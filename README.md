# 建筑工地施工隐患巡查智能体

基于 Vue3 + TypeScript + Ant Design Vue 的建筑工地施工隐患巡查管理系统。

## 技术栈

- **框架**: Vue 3.4+ (Composition API)
- **构建工具**: Vite 5.x
- **语言**: TypeScript
- **UI组件库**: Ant Design Vue 4.x
- **状态管理**: Pinia
- **路由**: Vue Router 4.x
- **CSS预处理器**: SCSS
- **图表库**: ECharts 5.x
- **日期处理**: Day.js

## 功能模块

### 1. 首页（Dashboard）
- 今日巡检任务统计
- 隐患数量统计（发现、已整改、待处理）
- 近7天巡检趋势图
- 隐患类型分布饼图
- 最新动态列表

### 2. 智能巡检
- 巡检任务列表展示
- 多条件搜索（区域、类型、状态）
- 新增巡检任务
- 巡检任务操作（开始、完成、查看、删除）
- 数据导出功能

### 3. 隐患管理
- 隐患记录列表
- 多条件筛选（区域、类型、风险等级、状态）
- 新增隐患记录（含照片上传）
- 隐患详情查看
- 隐患处理流程（整改、验收）
- 批量处理功能

### 4. 统计分析
- 核心指标展示（巡检次数、隐患数、整改率、响应时间）
- 月度巡检趋势柱状图
- 隐患类型分布环形图
- 各区域隐患统计堆叠柱状图
- 风险等级仪表盘
- 详细数据报表及导出

### 5. 系统管理
- **用户管理**：用户CRUD、角色分配、状态管理
- **角色权限**：角色管理、权限配置
- **系统设置**：基本设置、通知配置
- **操作日志**：系统操作记录查询

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发环境运行

```bash
npm run dev
```

访问 http://localhost:5173

### 生产环境构建

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 主要特性

✅ **现代化UI设计**：采用Ant Design Vue组件库，界面美观专业  
✅ **响应式布局**：支持PC端和移动端自适应  
✅ **TypeScript支持**：完整的类型定义，提升开发体验  
✅ **模块化架构**：清晰的目录结构，易于维护和扩展

## 页面截图说明

系统采用蓝色渐变主题色，符合建筑行业信息化系统的视觉风格：

- **顶部导航栏**：蓝色渐变背景，包含Logo、系统名称、通知和用户信息
- **侧边栏菜单**：深色主题，5个一级菜单项，支持折叠展开

## 注意事项

1. 本项目为前端演示版本，使用模拟数据
2. 实际使用时需要对接后端API接口
3. 图片上传功能需要配置存储服务
4. 建议根据实际需求调整权限控制逻辑

## 开发规范

- 使用 Composition API 编写组件
- 组件命名采用 PascalCase
- 变量命名采用 camelCase
- 常量命名采用 UPPER_SNAKE_CASE
- 所有代码添加适当的注释
